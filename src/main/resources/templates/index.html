Beer Next Door est un projet d'application web permettant à ses usagers de localiser autour d'eux des détaillants vendant des bières artisanales.

En se connectant à Beer Next Door, l'usager est localisé et celui-ci peut consulter les étalages des magasins alentours.

BND: API est l'interface d'administration de la base de données de Beer Next Door.

Documentation

La documentation présente les routes d'API de Beer Next Door.

Les requêtes en GET ne nécessitent pas de clef d'API.

En revanche, une clef d'API est nécessaire pour ajouter/modifier des entrées à la base de données. Cette clef vous est fournie en vous inscrivant à BND: API. Vous pouvez la retrouver sur votre page profil.

Types de bières

@GetMapping("types")

Format de réponse : une liste d'entités Type.

                    
                        [
                        {
                            "idType": 1,
                            "nameType": "Pilsner"
                        },
                        {
                            "idType": 2,
                            "nameType": "Stout"
                        }
                        ]

@GetMapping("types/{idType}")

Format de réponse : une entité Type.

                    
                        [
                        {
                        "idType": 1,
                          "nameType": "Pilsner"
                          }
                        ]

@PostMapping("{apiKey}/types")

Format d'envoi : une entité Type sans ID.

                    
                        [
                        {
                        "nameType": "New BeerType"
                        }
                        ]

@PutMapping("{apiKey}/types/{idType}")

Format d'envoi : une entité Type sans ID.

                    
                        [
                        {
                        "nameType": "New BeerType Modified"
                        }
                        ]

@DeleteMapping("{apiKey}/types/{idType}")

Format de réponse : un booléen

  • true
    La suppression est effective. 
  • false
    La suppression n'a pas été effectuée.
